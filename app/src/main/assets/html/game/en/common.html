Game process

the Game consists of set of planets of the system chosen in installations. The number of planets depends on system, for example, in the main system classic their 30 pieces. The purpose of a game is in exempting all planets of system from monsters, using for this purpose make-shifts, see Game elements
Starting a game, you appear on the planet on which have stopped last time, or if you play for the first time, you begin with the first. If in installations have changed system, then the planet on which you have stopped in this system is started.
