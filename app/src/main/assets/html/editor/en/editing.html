For a choice of an element of editing it is necessary to click it in a panel. The selected element will be selected with the frame. I.e. the frame shows what element is the active.
All elements of editing settle down in three lines / a column (depending on orientation of the device) on a panel. Each of them makes the following sense:
  1. Stock
  2. Environment
  3. of a Being
All these elements are described in the help information on a game.
Installation of an element
After separation of the required element, it can be set on the planet.
For this purpose it is necessary to click the point interesting you and the element will be written in it. If not to release contact from the screen and its message - the element will be placed in all line items of the planet, corresponding to contact points.
If on a panel it is selected a element of a being, then installation of an element becomes complicated a little.
If it is selected by , then the previous line item of the droid will be erased (if was) and replaced with current position of contact, with restoration of that element, which was under it before contact. I.e. the planet can have a droid only of one!.
If in a contact line item, there is already a droid - there will be a changeover of his direction, see below. In a consequence in case of start of the planet in a game, at first that part of the planet where it was set by will be shown
If it is selected by , then installation of an element on the same element on the planet will lead to changeover it on and vice versa.
In case of installation of other beings, the principle such. Each being has four directions of movement, for example . Therefore to involve this mechanism, it is necessary to set an element of a being, on same in a point of contact and then, the element with an increment in the direction will be set.
Navigation
If only the part of the planet is displayed, it is possible to change this visible part. For this purpose it is necessary to use so-called long click
Having clicked on an every spot on the globe and retaining contact certain time (without moving a finger) the editor will pass into the shift mode. This mode will visually be designated by display blackening. Without releasing contact move it in the necessary napraleniye - the planet will move. In case of release the editor will return to a former status again.
For an output from the editor press back button on the device. If the planet was changed, that, the editor will ask you about need of saving changes.
