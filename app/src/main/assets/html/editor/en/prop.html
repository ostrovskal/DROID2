planet Parameters
If after creation of the planet, errors were noticed, it is possible, having used this window to change them. It is possible to change all parameters, except width and height of the planet and also like filling - the planet is already created and filled. If it is necessary to change filling type - then at first delete it, see the Interface and create again.
It is possible to change the following characteristics:
Number a new line item of the planet in system.
a Name a new unique name.
of fuel new volume
Time on-new necessary
System the planet will be transferred to other system.
