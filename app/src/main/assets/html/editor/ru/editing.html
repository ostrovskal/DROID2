Редактирование
Элементы редактирования
Для выбора элемента редактирования надо кликнуть по нему в панели. Выбранный элемент будет выделен рамкой. Т.е. рамка показывает какой элемент является активным.
Все элементы редактирования располагаются в три строки/столбца(в зависимости от ориентации устройства) на панели. Каждый из них имеет следующий смысл:
  1. Окружение
  2. Инвентарь
  3. Существа
Они описаны в справочной информации по игре.
Установка элемента
После выделения требуемого элемента, его можно установить на планете.
Для этого необходимо кликнуть по интересующей тебя точке и элемент будет записан в ней. Если не отпускать касание от экрана и вести его - элемент будет помещен во все позиции планеты, соответствующему точкам касания.
Если на панели выбран элемент типа существа, то установка элемента немного усложняется.
Если выбран , то предыдушая позиция дроида будет стерта(если была) и заменена на текущую позицию касания, с восстановлением того элемента, который был под ним до касания. Т.е. дроид у планеты может быть только один!.
Если в позиции касания, уже есть дроид - произойдет замена его направления, см. ниже. В последствии при запуске планеты в игре, сначала будет показана та часть планеты, где был установлен
Если выбран , то установка элемента на такой же элемент на планете приведет к замене его на и наоборот.
При установке других существ, принцип такой. У каждого существа существуют четыре направления движения, например . Поэтому чтобы задействовать этот механизм, надо устанавливать элемент существа, на такой же в точке касания и тогда, устанавливатся будет элемент с инкрементом по направлению.
Навигация
Если отображается только часть планеты, можно изменять эту видимую часть. Для этого надо использовать так называемый долгий клик
Кликнув на любой точке планеты и удерживая касание определенное время(не двигая палец) редактор перейдет в режим сдвига. Этот режим визуально обозначится потемнением отображения. Не отпуская касание двигайте его в нужном напралении - планета будет сдвигаться. При отпуске редактор вновь вернется в прежнее состояние.
Для выхода из редактора нажмите кнопку back на устройстве. Если планета была изменена, то, редактор спросит у вас о необходимости сохранения изменений.
