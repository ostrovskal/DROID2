Загрузка планеты
Все планеты доступные для загрузки представлены в виде списка, отсортированном по позиции планет в системе.
В каждом элементе списка показаны:
Превью планеты
Название
Дата создания
А также две стрелки вверх/вниз. Стрелки позволяют налету менять порядок следования планет в системе.
В окне загрузки можно выбирать предустановленные системы планет. В этом случае, при выборе системы, будут показаны планеты из нее.
По умолчанию выбрана система в которой находится планета, редактируемая в данный момент.
При клике на интересующей вас планете, она будет загружена и активирована для редактирования.
