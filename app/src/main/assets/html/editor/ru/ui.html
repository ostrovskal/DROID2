Интерфейс

Экран редактора планет и систем визуально разделен на две части:
  • В основной находится загруженная планета(если есть), в которой и происходит непосредственно процесс редактирования, см. Редактирование
  • Во-второй части панель управления состоянием редактора и выбора элементов редактирования.
Меню управления
Для вызова управляющего меню, необходимо кликнуть по шапке панели. Это активирует вызов формы управления редактором.
Значки в форме управления имеют следующий смысл:
Загрузка планеты
Создание новой планеты
Параметры планеты
Сохранение планеты, при ее изменении. Доступно только, если планета была изменена.
Отображение превью всей планеты, вписав ее в основную часть экрана редактора. По умолчанию отображается часть планеты, которая помещается в экран в соответсвии с настройками масштабирования. Эта команда как раз и меняет это поведение, либо отображает всю планету, либо показыает только ее часть. На устройствах с маленьким экраном редактирование при отображении все планеты будет затруднительным, поэтому и был введен данный функционал.
Перегенерация планет системы CLASSIC. Бывает необходимо при порче системы, либо, недопустимых изменениях
Удаляет текущую планету из системы и редактора.
Запускает режим тестирования редактируемой в данный момент планеты
Вызывает форму, позволяющую задать параметры, для отправки системы планет(кроме CLASSIC) на сервер
