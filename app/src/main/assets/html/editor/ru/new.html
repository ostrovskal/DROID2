Создание планеты
При создании новой планеты, необходимо задать несколько параметров.
Содержимое планет формируется случайным образом, по определенному алгоритму. Чтобы воздействовать на этот процесс, необходимо выбрать в списке этого окна тип заполнения. Этот тип будет создавать преимущество в заполнение определенным элементом.

Следующем этапом, необходимо определить систему, в которую будет помещена планета. Или создать новую - выбираем "..." и задаем имя системы. Имя должно быть уникально и не пересекаться с другими системами установленными на устройстве. По умолчанию выбрана система, выбранная в предыдущий раз или заданная в установках игры.
Характеристика планеты
Далее на экране создания планеты задаем следующие параметры:
Номер Позиция в системе. Необходим для определения того, какая планета следует за другой.
Название Уникальное имя. Не должно пересекаться с другими именами планет в выбранной системе.
Топливо Объем топлива, которое по вашему мнению хватит дроиду для освобождения планеты.
Время Необходимое время, за которое следует успеть освободить планету.
Высота и ширина Задают габариты планеты.
После внесения данных величин и проверки их на корректность, редактор создаст для вас эту планету и перейдет в режим редактирования, см. Редактирование
Кстати, вам необходимо знать, что пустые системы, в которых нет ни одной планеты, при запуске игры, автоматически удаляются...
